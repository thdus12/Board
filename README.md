# 게시판

구성환경 : SpringBoot, Gradle, Thymeleaf, Jpa(JPQL), Jar, MariaDB

## 개요

SpringBoot를 이용한 게시판 프로젝트 작성

### ✅ 로그인, 회원가입

스프링 시큐리티(Spring Security)를 이용한 로그인 처리, 로그아웃, 중복 로그인 처리, 로그인 정보 유지, 예외처리 등을 적용
